# Naming Conventions

## General Rules

*  No numbers or special characters in names (excepting underscores)
*  No 1 letter variable names / magic numbers
*  Try to make names descriptive and somewhat concise

## File Names

### Code Files: 
    PascalCase.js

### Asset Files: 
    lowercase.png

## JavaScript

### Class Names: 
    PascalCase

### Function Names: 
    camelCase()

### Function Params: 
    under_scored_lower

### Local Variables: 
    var under_scored_lower

## HTML

### Div Id / Class Names: 
    <div id="hyphen-lower"></div>

## Vue

### Component Names: 
    PascalCase

### Event Names: 
    this.$emit('hyphen-lower');
